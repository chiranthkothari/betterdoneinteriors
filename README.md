# Better Done Interiors

A re-creation of [betterdoneinteriors.com](https://betterdoneinteriors.com/) — a static, responsive
multi-page website for a Bangalore interior design studio.

## Pages
- `index.html` — Home
- `about.html` — About Us
- `services.html` — Services
- `portfolio.html` — Portfolio
- `contact.html` — Contact

## Stack
Plain HTML, CSS (`assets/css/style.css`) and vanilla JS (`assets/js/main.js`).
Fonts: Fraunces + Manrope. No build step.

## Local preview
Open `index.html` in a browser, or serve the folder:
```
python3 -m http.server
```
